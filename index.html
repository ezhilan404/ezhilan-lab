About Me

        

        

          

            
Who am I?

            

              I’m a student just starting my journey into ethical hacking, with a growing obsession for
              network security and red-team style thinking. I like to see what really happens behind the
              screens – the packets, protocols, and hidden paths that most people never think about.
            

            

              I’m not here to just run tools, I want to understand them. I learn by building things from
              scratch instead of relying only on ready-made tools, because fundamentals matter more than
              shortcuts. Long term, my goal is simple: get really good at ethical hacking and turn this
              curiosity into a career in cybersecurity.

Security Projects

        

        

          

            
Port Scanner

            
python · sockets · networking

            

              A Python-based port scanner that discovers open ports on a target host and simulates some of
              the behavior of tools like nmap. Built to understand TCP/IP, sockets, and basic recon.
            

            

              recon
              networking
            

          

          

            
Firewall Simulator

            
python · rules engine

            

              Simulates how firewalls process packets against rule sets. I experiment with allow/deny rules,
              logging, and how misconfigurations can create security gaps.
            

            

              defense
              traffic filtering
            

          

          

            
Honeypot

            
python · sockets

            

              A honeypot that listens on specific ports, logs incoming connections, and lets me study
              attacker behavior in a controlled environment.
            

            

              deception
              logging
            

          

          

            
Keylogger Detector

            
python

            

              A detection-focused project to spot suspicious keylogging activity based on basic behavior
              patterns. Helped me understand how keyloggers work under the hood — and how to fight them.
            

            

              detection
              host security
            

          

          

            
Password Strength Analyzer

            
python

            

              Scores password strength based on length, character variety, and patterns. A simple project
              with a big impact on understanding real-world password weaknesses.
            

            

              defense
              auth
            

          

          

            
Web Vulnerability Scanner

            
python · http

            

              A basic scanner that checks web targets for common security misconfigurations and simple
              vulnerabilities, giving me hands-on experience with web app security.
            

            

              web security
              automation
